# NorthStar Urban Mobility Analytics

A full-stack data analytics case study investigating operational failures 
across a fictional urban logistics company using SQL, R, Python, and MongoDB.

## Project Overview
NorthStar is experiencing rising delivery failures, driver incidents, and 
customer complaints. This project diagnoses root causes and builds a 
predictive model for delivery outcomes across 9 relational datasets 
(30,000+ records).

## What's Covered
- **Section 1–2 (R):** SQL queries inside R using sqldf — JOINs, 
  aggregations, zone-level KPIs, ggplot2 visualisations
- **Section 3 (Python):** Data cleaning, EDA (heatmaps, pairplots), 
  feature engineering, Random Forest & Logistic Regression classifier 
  to predict delivery failure
- **Section 4 (MongoDB):** NoSQL schema design, MongoDB Atlas connection, 
  CRUD operations on embedded document collections

## Tools & Technologies
Python, R, SQL, MongoDB Atlas, Pandas, NumPy, Scikit-learn, 
Seaborn, Matplotlib, ggplot2, PyMongo

## Dataset
9 CSV files covering: orders, deliveries, drivers, vehicles, 
hubs, customers, complaints, incidents, app events
